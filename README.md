A Python-based mini project to detect fake and real QR codes in payment systems.

---

## 📌 Objective

To improve digital payment security by identifying fraudulent QR codes before payment is processed.

---

## 🚀 Features

- Random QR generation  
- Fake QR detection  
- Real QR payment simulation  
- OTP generation  
- OpenCV scanning animation  
- GitHub version control  

---

## 🛠 Technologies Used

- Python  
- OpenCV  
- PyOTP  
- QRCode  
- NumPy
